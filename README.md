# Smart Medical Expense Manager

🚀 **SMART MEDICAL EXPENSE MANAGER**  
“Turning confusing hospital bills into smart financial decisions”

## 🧠 Problem Statement

Medical expenses are:

- **Confusing** (complex bills, hidden charges)
- **Unplanned** (emergencies = financial stress)
- **Inefficient** (people overpay without knowing alternatives)

👉 People don't know:

- What they are paying for
- If it's overpriced
- If insurance covers it
- If cheaper options exist

## Features

- 📸 **Scan Bills**: Upload medical bills and extract text using OCR.
- 🔍 **Compare Prices**: Compare prices for medical procedures across hospitals.
- 📊 **Dashboard**: Visualize expense categories with charts.
- 🤖 **AI Chatbot**: Get instant advice on medical expenses.
- 🧾 **Categorize**: Break down bills into categories.
- 💰 **Save Money**: Suggestions for cost-saving.
- 🛡️ **Insurance & Payments**: Help with claims and payment options.
